# PET Grammar Lab - Adjectives & Comparisons

Website tĩnh cho học sinh A2-B1/PET luyện:

- Gradable and Non-gradable Adjectives
- Comparative Adjectives
- Superlative Adjectives
- as...as / not as...as
- more / fewer / less
- PET Speaking/Writing examples
- Bài tập tương tác, chấm điểm tức thì, feedback tiếng Việt

## Cách chạy trên máy

Mở file `index.html` bằng trình duyệt Chrome, Edge hoặc Safari.

## Cách đưa lên GitHub Pages

1. Tạo repository mới trên GitHub.
2. Upload toàn bộ các file trong thư mục này lên repository:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Vào **Settings** → **Pages**.
4. Ở mục **Build and deployment**, chọn:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Bấm **Save**.
6. Sau vài phút, GitHub sẽ cung cấp link dạng:
   `https://USERNAME.github.io/REPOSITORY-NAME/`

## Lưu ý

Trang này không cần server hoặc database. Kết quả được lưu tạm trên trình duyệt của học sinh bằng localStorage. Học sinh có thể bấm nút “Copy kết quả gửi giáo viên”.
